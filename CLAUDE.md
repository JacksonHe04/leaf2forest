# Leaf2Forest

Leaf2Forest 是安徽省青阳中学 2019 级 2 班的数字档案馆，用于长期保存班级成员信息、成长轨迹以及高中时期珍贵的声音记录。

* Leaf：代表每一位同学，每个人都是独立成长的一片叶子。
* Forest：代表整个班级，60 多位同学共同构成一片森林。

它记录：

* 每个人现在在哪里；
* 每个人正在做什么；
* 每个人从哪里出发；
* 那段共同经历过的高中岁月留下了什么声音。

## 产品设计

### 1. Classmates（同学档案）

记录每一位同学当前状态：基本信息、所在城市、当前学校、工作单位、所属行业、个人介绍、联系方式等

### 2. Echoes（声音档案）

保存高中时期录制的 200+ 条音频。这些音频不是普通媒体文件，而是：一个时代留下来的声音样本。每一条录音应该包含：时间、地点、背景、参与人物、内容描述、转录文本等。让未来打开网站的人，可以重新回到那个时间节点。

| 关键词     | 解释       |
| ------- | -------- |
| Warm    | 温暖，不冰冷   |
| Memory  | 记忆与回忆    |
| Growth  | 成长轨迹     |
| Forest  | 自然隐喻     |

## 权限系统

### 游客

默认访问者。
* 查看首页
* 查看班级信息
* 浏览同学档案
* 浏览录音档案
* 播放公开音频

禁止：
* 修改任何内容
* 上传内容

### 已登录同学

通过用户名 + 密码登录。登录目的：不是为了限制访问，而是为了允许本人维护资料。
* 编辑自己的个人信息
* 修改头像
* 更新个人介绍
* 更新学校/工作信息

禁止：
* 修改其他同学资料
* 删除档案
* 修改录音

### 管理员

项目维护者。
* 所有同学信息
* 所有录音信息
* 图片资源
* 音频资源
* 用户账号
